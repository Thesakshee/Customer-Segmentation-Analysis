# Customer Segmentation Analysis (SQL + Power BI) 👥

### Project Overview
This project focuses on behavioral analytics by segmenting customers into distinct groups based on their purchasing patterns. By applying **RFM (Recency, Frequency, Monetary)** principles, I identified high-value "VIP" customers and at-risk segments to provide data-driven marketing recommendations.

### Tech Stack
* **Database:** SQL (Advanced CTEs & CASE Logic)
* **Visualization:** Power BI
* **Data Modeling:** Power Query (Data Cleaning & Transformation)

### Key Insights
* **The 80/20 Rule:** Discovered that 22% of customers (VIP Segment) drive nearly 60% of total revenue.
* **Churn Risk:** Identified a segment of "At-Risk" customers who haven't purchased in 6+ months, representing a $1.2M revenue recovery opportunity.
* **Product Preference:** The "Enterprise" segment prefers high-volume, low-frequency orders, while "Small Business" segments show high frequency with lower margins.
